Immigration Setup

The Immigration Setup dialog allows you to define the profile of Varroa infestation. You can select from one of six immigration types, enter how many mites in total to immigrate, and identify the immigration period.

Immigration Types

There are six predefined immigration types corresponding to a distribution of Varroa entering the colony over the immigration duration. You can select a particular immigration type by clicking on the example graph. The immigration type you have currently selected is shown in the box labeled “Immigration Type Selected”.

Number of Mites to Immigrate

This box contains the number of mites you would like to simulate immigrating into the colony.

Immigration Start and Stop Dates

These are the dates that immigration begins and ends. These dates are constrained to be within the dates of the simulation start and stop dates.

Completing Immigration Setup

When you have set up immigration to your satisfaction, press the OK button to accept the changes. If you would like to ignore the changes you made, press the Cancel button.
