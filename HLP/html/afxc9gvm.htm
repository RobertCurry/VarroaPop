Create A Weather File

Weather files in VarroaPop are plain text files with predefined columns and a predefined header. The easiest way to create a new weather file is to modify one of the existing ones in some text editor. If you use a word processor to edit the file, be sure to save it in plain text format. Weather files must have a “.wth” extension to be used by VarroaPop. Weather files must also be located in the same directory as the VarroaPop executable file.
