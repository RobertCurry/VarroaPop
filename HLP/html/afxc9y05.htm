Save command (File menu)

Use this command to save the active session or the simulation results by selecting “Session” or “Results” from the pop-up menu.

Save Session:

When you save a session for the first time, VarroaPop displays the Save As dialog box so you can name your session. If you want to change the name and directory of an existing session before you save it, choose the Save As command.

Save Results:

If you choose to save results, a Save As dialog box is shown so you can name your results file. The default extension is “.res”. This file is saved as normal space-delimited text and can be easily imported into spreadsheets or word processors.

Shortcuts

Toolbar:

Keys:CTRL+S
