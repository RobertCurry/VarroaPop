Select Parameters to Plot

This dialog box allows you to select which of the tracked parameters you would like to plot. You can select anywhere from zero to all parameters however more that 4 or 5 parameters makes the plot harder to read. Note that the values of proportion of infestation will range from 0 to 1.0 while all other parameters can get much larger. Consequently, if you plot proportions and populations on the same graph, the scaling is likely to be such that the proportion data will not be meaningful visually.
