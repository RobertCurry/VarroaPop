Getting Started

VarroaPop is a simulation tool that allows you to predict the behavior of honeybee colonies that are infested with Varroa mites. The power of VarroaPop lies in its ability to vary a wide range of initial colony conditions, weather, and mite immigration profiles.

When you first begin VarroaPop, you will see a menu at the top with File, View, and Help items. A shortcut toolbar is just below the menu. You will also see some controls along the left side labeled Immigration and Simulation Period

A simulation session is begun by either loading an existing session or by creating a new session. A new session is created in one of two ways. The first way is by either the File menu item then selecting the New menu item This will display the Model Parameters dialog box that can be filled out with colony initial conditions and the desired weather data file. The second way to create a new session is to press the New Session button on the toolbar. This method will also display the Model Parameters dialog box.

Once a session is loaded, you can run a simulation . When your simulation run has completed, you will get both a graphical display and a tabular display. The graphical display can be saved or printed by selecting “Send” then either “To File” or “To Printer” on the graphical display menu. The tabular display data can be saved or printed by selecting Save or Print from the File Menu
