How to Run a Simulation

After you have loaded an existing session or created a new one, you are ready to run a simulation. On the left side of the screen is a box labeled “Simulation Period”. This box contains editable fields for simulation start and stop dates. You can set these to any dates within the scope of the current weather file. There is also a box labeled “Immigration”. You can setup the Immigration profile by pressing the Setup button in and you can enable or disable immigration by clicking on the proper button.
