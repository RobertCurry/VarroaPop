Model Parameters

The Model Parameters screen is where the initial conditions for the colony are displayed and edited, where the Weather file selection is made, and where actions to the colony are specified.

There are three tabs on the Model Parameters screen: Colony Data, Weather Data, and Actions. The information on these screens is used to control the simulation.

Once you have entered the desired information in each of the tabbed pages, press the OK button to accept these changes or press the Cancel button to ignore them.
